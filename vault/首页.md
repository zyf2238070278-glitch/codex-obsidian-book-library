# 本地书库首页

这个书库可以完全通过 Codex 对话使用。更完整的目录约定与故障说明请见 [[书库/说明]]。

## 第一次使用

打开本项目时，请先在 Codex 中信任项目。新 MCP 配置首次加入后，需要重新加载 Codex 一次；以后导入、检索和保存笔记都可以留在对话窗口完成。

## 可以直接这样说

- 在 Codex 中上传书籍：附加 PDF、EPUB、Markdown 或 TXT，然后说“把这本书导入书库”。Codex 会使用附件的本机绝对路径。
- 列出书库：“列出书库里的书和处理状态。”
- 引用原文：“查找《书名》里关于库存周期的原话，引用原文并标出位置。”
- 通俗解释：“根据书中证据，用通俗解释说明为什么供给会滞后。”
- 跨书比较：“跨书比较这几本书对同一问题的看法，分别给出依据。”
- 保存到 Obsidian：“把刚才的回答连同原文依据保存到 Obsidian，标题叫……”。只有明确提出保存时才会写笔记。

完整书籍和索引保留在本机；普通回答只把本次检索到的少量证据交给 Codex。
